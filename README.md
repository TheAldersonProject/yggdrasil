# Yggdrasil

## Project Name

Yggdrasil (from Old Norse Yggdrasill) is an immense and central sacred tree in Norse cosmology.

## Project Description

Python implementation for the [**Open Data Contract Standard**](https://bitol-io.github.io/open-data-contract-standard/latest/).

## Purpose

Create and manage data contracts assets, following the ODC Standard.

## Implementation Tools

- Python 3.10+
- UV: For package management
- Pydantic: To implement the ODC Standard
- Structlog: For structured logging
- SQLite: For data contract schema validation
- DuckDB: For ontology management

## Implementation

This project relies on a specific implementation proposal, which includes the following steps:

1. Pydantic models for the ODC Standard. 
2. YAML templates for the ODC Standard. These templates are examples following the ODC Standard.
3. Implement validation of the pydantic models against the YAML templates.
4. Implement a metadata management system.
